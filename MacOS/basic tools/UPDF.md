## UPDF 是什么

UPDF 是一款好用且不贵的国产 PDF 工具。

- 官网：https://updf.com/
- 中文官网：https://www.updf.cn/
- `brew install --cask updf`

## UPDF 能做什么

- PDF 编辑和标注
- 扫描版 PDF 识别文字并导出
- 给 PDF 瘦身（压缩）
- UCloud 云存储同步
- PDF 转换导出 Office、图片等格式
