## 学术垃圾生产的传统方式

### 复制与粘贴

复制与粘贴，是指通过重用已有的资料来充实自身文章的方法。例如在计算机领域常见的CV大法：

```c++
#include <iostream>
using namespace std;
int main()
{
    cout<<"Hello, World!"<<endl;
    return 0;
}
```

### 废话与扩写

废话与扩写，是指在已有的文句中插入看似相关实则可有可无的名人名言，或以近义词替换的形式在不改变句子原有含义的情况下增加长度。例如歌德曾说过：读一本好书，就如同和一个高尚的人在交谈[^Goethe]。
